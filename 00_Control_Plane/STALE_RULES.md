# Stale Rules

1. AI memory is non-authoritative.
2. Old drafts are non-authoritative.
3. Stakeholder files are derived views only.
4. Context packs are derived views only.
5. If a file conflicts with `NOW.md`, use `NOW.md` for current status.
6. If a file conflicts with a formal decision log, use the formal decision log.
7. Do not delete stale history. Mark it `SUPERSEDED`, `CLOSED`, `RETIRED`, or `HISTORICAL ONLY`.
8. Any open request older than 14 days must be revalidated before execution.
