# Folder Purpose: GPT_to_ClaudeCode

This folder stores GPT Handoff Patches authored by ChatGPT for Claude Code to execute.

Filename convention:
`YYYY-MM-DD_<short-topic>_vNN.md`

Use the template at `_GLOBAL_PROTOCOLS/GPT_HANDOFF_PATCH_TEMPLATE.md`.

Do not delete old handoffs. Move completed/applied patches to `../Applied/` or mark their status in the file.
