# Folder Purpose: ClaudeCode_Feedback_To_GPT

After executing a GPT Handoff Patch, Claude Code writes a feedback file here for ChatGPT to review.

Filename convention:
`YYYY-MM-DD_Feedback_<short-topic>_vNN.md`

The feedback must include:
- files modified;
- summary of changes;
- ledger updates;
- stakeholder view updates;
- context pack updates;
- deviations from the GPT handoff patch;
- issues discovered;
- git diff summary;
- commit hash, if committed;
- push status, if pushed;
- explicit request for ChatGPT review.
