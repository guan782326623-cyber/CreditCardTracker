# Folder Purpose: ClaudeCode_PreExecution_Review

Before executing any GPT Handoff Patch, Claude Code writes a pre-execution review here.

Filename convention:
`YYYY-MM-DD_PreExec_<short-topic>_vNN.md`

The review must include:
- the GPT handoff patch reviewed (link);
- the current source-of-truth files checked;
- Claude Code's understanding of the task;
- agreement / disagreement with GPT;
- files proposed for modification;
- ledger IDs affected;
- risks or open questions;
- whether Teddy approval is required.

Claude Code must wait for Teddy approval before executing.
